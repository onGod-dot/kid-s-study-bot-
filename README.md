# Buddy - AI Study Companion

An intelligent study companion designed for kids of all ages, featuring age-appropriate interfaces, AI-powered tutoring, and comprehensive learning tools.

## Features

### 🌲 Kids Interface (Ages 5-12)
- **Forest Theme**: Engaging woodland environment with friendly animal companions
- **Interactive Chatbot**: Choose from forest animals (Owl, Monkey, Rabbit, Fox, Deer, Bear)
- **Voice Interaction**: Click to talk with your AI companion
- **Assignment Helper**: Upload homework images for AI-guided assistance
- **STEM Learning**: Age-appropriate science, technology, engineering, and math activities

### 🚀 Teens Interface (Ages 13-18)
- **Space Theme**: Cosmic learning environment with celestial mentors
- **Advanced Chatbot**: Choose from space beings (Star, Astronaut, Alien, Planet, Comet, Nebula)
- **Sophisticated Learning**: Advanced problem-solving and critical thinking
- **Complex Assignments**: Handle advanced homework and projects
- **Advanced STEM**: Robotics, coding, data science, and game development

### 👨‍👩‍👧‍👦 Parent Dashboard
- **Progress Tracking**: Monitor your child's learning journey
- **Activity Analytics**: Detailed insights into learning patterns
- **Time Management**: Track study time and productivity
- **Skill Development**: See what your child is learning
- **Safety Monitoring**: Ensure appropriate content and usage

## Technology Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS with custom themes
- **Animations**: Framer Motion
- **AI Integration**: Together AI API
- **State Management**: Zustand
- **File Handling**: React Dropzone

## AI Capabilities

### Together AI Integration
- **Chat Completions**: Intelligent conversations with age-appropriate responses
- **Speech-to-Text**: Voice input processing for hands-free interaction
- **Image Generation**: Create visual learning aids and illustrations
- **Assignment Analysis**: AI-powered homework assistance

### Learning Approach
- **Guided Discovery**: Never gives direct answers, guides students to solutions
- **Age-Appropriate**: Different complexity levels for different age groups
- **Interactive Learning**: Engaging conversations and activities
- **Progress Tracking**: Monitors learning development over time

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd buddy-ai-study-companion
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
# Create .env.local file
TOGETHER_API_KEY=your_together_ai_api_key_here
```

4. Run the development server
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

### For Kids (Ages 5-12)
1. Select "Kids" from the main menu
2. Choose your favorite forest animal companion
3. Start chatting or upload homework for help
4. Explore STEM activities designed for your age

### For Teens (Ages 13-18)
1. Select "Teens" from the main menu
2. Choose your cosmic mentor
3. Engage in advanced learning conversations
4. Work on complex STEM projects

### For Parents
1. Select "Parents" from the main menu
2. Monitor your child's progress
3. View learning analytics and time spent
4. Track skill development and achievements
